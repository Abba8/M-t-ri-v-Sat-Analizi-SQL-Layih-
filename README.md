# M-t-ri-v-Sat-Analizi-SQL-Layih-
Bu layihədə Northwind database-dən istifadə edərək aylıq satışların analizini göstərən SQL sorğuları hazırlanmışdır. Layihə satış miqdarı, məhsul qiymətləri və sifariş məlumatlarını birləşdirərək müxtəlif statistik nəticələr çıxarmağa imkan verir.


Əsas xüsusiyyətlər:

Müxtəlif SQL sorğuları ilə satış məlumatlarının çıxarılması

Aylıq və məhsul üzrə satışların analizi

Qruplaşdırma, aggregate funksiyalar və JOIN-lərdən istifadə

Analitik və təhlil bacarıqlarının nümayişi

İstifadə olunmuş texnologiya:
PostgreSQL
